> Projeto simples de um sistema de carrinho de compras inspirado na Shopee, desenvolvido como prática em um curso de Node.js da DIO.

---

## 📌 Sobre o Projeto

Este projeto simula um carrinho de compras onde é possível:

- Adicionar produtos
- Remover produtos
- Deletar produtos do carrinho
- Visualizar os itens adicionados
- Calcular o valor total

O foco principal não é interface visual, mas sim **lógica de programação e organização do código**.

---

## 🎯 Objetivos do Projeto

- ✅ Reforçar conceitos de **modularização com ESM (ECMAScript Modules)**
- ✅ Praticar **lógica de programação**
- ✅ Trabalhar com **arrays e métodos como `find`, `reduce` e `forEach`**
- ✅ Organizar código em **services**
- ✅ Criar funções reutilizáveis

---

---

## ⚙️ Funcionalidades

- ➕ Adicionar produtos ao carrinho
- ➖ Remover uma unidade de um produto
- ❌ Deletar um produto completamente do carrinho
- 📋 Listar todos os produtos com detalhes (nome, preço, quantidade e subtotal)
- 💰 Calcular o valor total do carrinho

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/Carrinho-de-Compras-Shopee.git
```

2. Acesse a pasta do projeto.

```bash
cd Carrinho-de-Compras-shopee
```

3. Execute o Projeto:

```bash
node src/index.js
```
